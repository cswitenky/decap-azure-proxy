About

This is an Azure Static Web Apps GitHub OAuth proxy for Decap CMS. It allows deploying Decap CMS without requiring Netlify Identity or Git Gateway services.

Setup

Configure your Decap CMS config.yml to use this proxy:

backend:
  name: github
  branch: main
  repo: "your-username/your-repo"
  base_url: https://your-proxy-domain.azurestaticapps.net
